Widget zum Anzeigen von Gebrauchtmöbel, die gerade abgegeben werden.
 
== Description ==

Dieses Plugin enthält ein Widget, welches Gebrauchtmöbel anzeigt, die abgegeben werden möchten, ansonsten landen diese in der Regel im Sperrmüll. Es nutzt die frei zugängliche API von WeiterGeben.org mit Zugriff auf die komplette Datenbank mit zur Verfügung stehenden Gebrauchtmöbeln. Region und Möbelkategorie können individuell voreingestellt werden.

Von den 60 Millionen, jährlich vernichteten Altmöbeln in den deutschsprachigen Ländern, könnte ca. die Hälfte in die Weiternutzung gebracht werden. Zu diesem Zweck wurde dieses Widget ehrenamtlich erstellt. 

Durch Klick auf den Gegenstand, wird man zur jeweiligen Produktseite von WeiterGeben.org geleitet und hat dann die Möglichkeit sich die Kontaktdaten des Möbelbesitzers zusenden zu lassen.
